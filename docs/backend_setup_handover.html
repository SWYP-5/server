로컬 세팅 방법 (팀원용)

1
레포지토리 클론
git clone <레포 주소>
cd app_backend
2
MySQL DB 생성
mysql -u root -p < src/main/resources/sql/schema.sql
또는 MySQL에서 직접:
CREATE DATABASE IF NOT EXISTS app_db
  CHARACTER SET utf8mb4
  COLLATE utf8mb4_unicode_ci;
3
비밀번호 확인 — application-local.yml
spring:
  datasource:
    username: root
    password: 1234  # 다르면 여기만 수정
기본값 root / 1234. 본인 MySQL 비밀번호가 다르면 이 파일만 수정.
4
서버 실행
./gradlew bootRun
또는 IntelliJ에서 AppBackendApplication.java → ▶
헬스체크: http://localhost:8080/api/health
Swagger: http://localhost:8080/swagger-ui/index.html
